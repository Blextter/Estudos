Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>, que é uma tag não-semântica.

Nesta outra frase, há outro termo em destaque porém utilizando a tag semântica <strong>.

Itálico / Ênfase

Nesta frase existe um termo em itálico usando a tag não-semântica <i>.

Nessa frase existe um termo em ênfase utilizando a tag <em>, que é uma tag semântica.

Tamanho de Textos

Podemos criar textos pequenos utilizando a tag <small>. Para textos maiores devemos utilizar CSS.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado utilizando a tag <del>.

Texto Inserido

Podemos também marcar um texto como inserido para dar significar que foi adicionado depois utilizando a tag <ins>.

Texto Sobrescrito

Para inserir coisas do tipo "x2+30=y" por exemplo, utilizamos a tag <sup>.

Texto Subescrito

Para inserir coisas do tipo "H2O" por exemplo, utilizamos a tag <sub>.
